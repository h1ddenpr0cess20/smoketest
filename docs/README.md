# smoketest Documentation

smoketest is a focused, cross-provider coding assistant built around the
OpenAI Responses API. It is a tool for planning and shipping code, not a
person or a relationship.

## Safety and use policies

- [AI Output Disclaimer and Conditions of Use](ai-output-disclaimer.md)
- [Not a Companion](not-a-companion.md)
- [Security](security.md)

Please read these documents before using or deploying smoketest. In
particular:

- smoketest is not intended for children or child-directed use.
- smoketest is not a friend, companion, therapist, or romantic partner.
- AI output — including generated code — can be wrong, insecure, or
  inappropriate. Verify it before acting on it.
- Do not rely on smoketest for emergencies, crisis response, professional
  advice, or other safety-critical decisions.
- API keys and MCP server definitions are only as safe as what you enable;
  MCP tool calls run without a manual approval step, so only connect servers
  you trust.
