# smoketest brand assets

The smoketest mark pairs a geometric pine silhouette with a single smoke plume and a restrained ember underline. Use the supplied files as-is; avoid stretching, rotating, outlining, or adding effects to the mark.

## Quick picks

- App icon or avatar: `png/app-icons/smoketest-app-icon-light-512.png` or the dark variant
- Small UI mark: `png/marks/smoketest-mark-light-128.png` or `smoketest-mark-dark-128.png`
- Transparent mark: `svg/smoketest-mark-on-light.svg` or `smoketest-mark-on-dark.svg`
- README/site banner: `svg/smoketest-banner-light.svg` or `smoketest-banner-dark.svg`
- Social preview: `png/social/smoketest-social-card-light-1200x630.png` or the dark variant

## Contents

## Palette

| Role       | Light     | Dark      |
| ---------- | --------- | --------- |
| Background | `#e5e2d8` | `#171a18` |
| Tree       | `#1d2420` | `#eeeae0` |
| Smoke      | `#858880` | `#666b66` |
| Ember      | `#ad583f` | `#d2764e` |

Keep clear space around the mark equal to roughly one quarter of its width. Prefer the light assets on light surfaces and dark assets on dark surfaces; use the transparent variants only when the surrounding surface already provides enough contrast.
